# Data Availability

## Data Access
This project uses NLSY79 and NLSCYA public-use files distributed by the U.S. Bureau of Labor Statistics NLS program.

Required raw files:
- `nlscya_all_1979-2020.zip`
- `nlsy79_all_1979-2022.zip`

Download source:
- `https://www.nlsinfo.org/content/access-data-investigator/accessing-data-cohorts`

## Repository Policy
- Raw data are not stored in this repository.
- Intermediate and derived datasets are not stored in this repository.
- Users must acquire data directly from official NLS sources and place them in `data/raw/`.
