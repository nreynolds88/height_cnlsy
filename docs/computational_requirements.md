# Computational Requirements

## Software
- Stata 17 or newer (tested target: StataMP 19)
- Windows PowerShell

## Hardware
- Recommended: 16 GB RAM minimum for comfortable processing of merged panel extracts.
- Disk: allow at least 15 GB free space for raw zips, extraction artifacts, and logs.

## Runtime
- Current runtime is unknown until scripts are fully implemented.
- Pipeline logs are written to `results/logs/` for profiling and debugging.
